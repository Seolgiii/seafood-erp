# 검색 결과 자동 트리거

## 상태
❓ 미해결

## 한 줄 요약
URL 복원 시 검색 자동 실행 옵션 — 채택 여부 미정

## 결정 내용
재고/신청/수신함 검색 필터를 URL 쿼리로 동기화하는 기능은 구현됨.
사용자가 북마크 또는 공유받은 URL로 진입할 때 검색이 자동 실행되어야 하는지 결정 필요.
자동 실행 시 의도치 않은 결과 노출 가능, 수동 트리거 시 한 번 더 클릭 필요.
