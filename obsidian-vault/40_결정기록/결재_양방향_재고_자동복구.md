# 결재_양방향_재고_자동복구

## 상태
✅ 확정

## 한 줄 요약
승인 ↔ 반려 토글 시 재고 자동 복구 (입고 soft delete, 출고 +outQty 복구)

## 결정 내용
입고 반려: LOT 재고수량=0 + 보관처 비용 3필드 null. 출고 반려: 잔여수량/LOT재고 +outQty 복구 + 출고시점 비용 7필드 null. 재승인 시 createLotOnInboundApproval 재실행으로 자동 복원. soft delete로 LOT 보존.
