# C1 동일조건 자동합산 → FIFO 의존 체인

## 트리거
"같은 조건"(품목/규격/미수)이지만 다른 선박에서 온 입고가 발생할 때

## 흐름 (의사결정 의존 — 실제 데이터 흐름이 아닌 결정 분기)
1. 같은 품목/규격/미수의 LOT이 다른 선박/시점에서 입고 → [[30_모듈별_상세/입고_관리]]
2. **결정 분기 — 동일조건_자동합산**:
   - 자동 합산 채택 시 → 단일 LOT으로 합쳐 LOT별 재고에 누적 → [[30_모듈별_상세/LOT별_재고]]
   - 분리 채택 시 → 선박 정보까지 키로 구분, 각각 별도 LOT 유지 → [[30_모듈별_상세/LOT별_재고]]
3. 출고시점 평단가 산정 (FIFO 또는 평균) → [[30_모듈별_상세/출고_관리]]
4. 부자재 재고 확장 시 — 품목코드 추가 + 미수→사이즈로 폼 일반화

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/입고_관리]] → [[30_모듈별_상세/LOT별_재고]]: 합산 키 정의 (품목+규격+미수, 또는 +선박)
- [[30_모듈별_상세/LOT별_재고]] → [[30_모듈별_상세/출고_관리]]: FIFO 큐 또는 평단가 산정
