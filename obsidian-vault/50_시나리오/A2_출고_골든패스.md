# A2 출고 골든패스

## 트리거
작업자가 재고 조회로 LOT 검색 → 카트에 다건 담기 → 출고 신청 → 관리자 승인

## 흐름
1. 작업자가 재고 조회에서 조건 검색 → LOT 후보 표시 → [[30_모듈별_상세/재고_조회]]
2. 다건 LOT 카트에 담기 (수량·고객 입력) → [[30_모듈별_상세/출고_관리]]
3. 입력 sanitize + zod 검증 → [[30_모듈별_상세/입력_Sanitize_Schema]]
4. POST + X-Idempotency-Key → [[30_모듈별_상세/POST_Idempotency]]
5. 출고 레코드 생성 (펜딩) → [[30_모듈별_상세/출고_관리]]
6. 관리자 결재 수신함 검토 → [[30_모듈별_상세/결재_수신함]]
7. 승인 — 출고 멱등 가드 체크 (판매원가>0이면 차단) → [[30_모듈별_상세/출고_관리]]
8. LOT 재고수량 차감 + 입고관리.잔여수량 차감 → [[30_모듈별_상세/LOT별_재고]]
9. 출고시점 비용 7필드 스냅샷 (cost-calc) → [[30_모듈별_상세/보관처_비용_이력]]
10. 출고증 PDF 생성 → [[30_모듈별_상세/PDF_생성]]
11. PDF Vercel Blob 업로드

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/재고_조회]] → [[30_모듈별_상세/출고_관리]]: 카트 LOT 리스트 (lotId, qty, 고객)
- [[30_모듈별_상세/출고_관리]] → [[30_모듈별_상세/LOT별_재고]]: 차감량 (재고수량/잔여수량)
- [[30_모듈별_상세/LOT별_재고]] → [[30_모듈별_상세/보관처_비용_이력]]: 출고시점 cost-calc 입력 (입고시점·출고시점·누적냉장료 등)
- [[30_모듈별_상세/출고_관리]] → [[30_모듈별_상세/PDF_생성]]: 출고증 데이터 → PDF Buffer
