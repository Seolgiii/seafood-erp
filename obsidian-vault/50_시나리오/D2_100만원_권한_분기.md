# D2 100만원 권한 분기

## 트리거
지출 결의 amount > 1,000,000 시 권한 분기

## 흐름
1. 작성자가 지출 결의 폼 amount 입력 → [[30_모듈별_상세/지출결의]]
2. 클라이언트 권한 표시는 참고용 (서버에서 재검증) → [[30_모듈별_상세/지출결의]]
3. 서버 액션 진입 시 작업자 role 재조회 → [[30_모듈별_상세/서버_권한_검증]]
4. amount > 1,000,000 분기:
   - role === 'MASTER' — 즉시 승인 가능 → [[30_모듈별_상세/결재_수신함]]
   - role === 'ADMIN' / 'WORKER' — 승인 차단, 결재 대기로 → [[30_모듈별_상세/결재_수신함]]
5. 통합 테스트(`expense-permission.test.ts`)로 우회 차단 검증

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/지출결의]] → [[30_모듈별_상세/서버_권한_검증]]: amount + 작업자 ID
- [[30_모듈별_상세/서버_권한_검증]] → [[30_모듈별_상세/결재_수신함]]: 승인 가능 여부 + role
