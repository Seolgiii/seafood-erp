# A5 재고 조회 3단계 플로우

## 트리거
작업자가 재고 조회 페이지 진입 → 조건 검색 → LOT 수량 선택 → 견적 요약 → 직접 출고 신청

## 흐름
1. 작업자가 재고 조회 페이지 진입 → [[30_모듈별_상세/재고_조회]]
2. 조건 검색 (품목명 자동완성 + 보관처 + 규격) — 디바운스 300ms 후 URL 쿼리 sync → [[30_모듈별_상세/재고_조회]]
3. LOT 후보 결과 표시 (전체기간 토글 + 풀수량 체크박스) → [[30_모듈별_상세/LOT별_재고]]
4. LOT 수량 선택 (카트에 담기) → [[30_모듈별_상세/재고_조회]]
5. 견적 요약 단계 (출고가 합 + 수량) → [[30_모듈별_상세/재고_조회]]
6. 자체 BulkSubmitSheet으로 직접 출고 신청 → [[30_모듈별_상세/출고_관리]]
7. (옵션) QR 스캔 진입 시 1단계로 LOT 자동 prefill → [[30_모듈별_상세/QR_스캔]]

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/LOT별_재고]] → [[30_모듈별_상세/재고_조회]]: 검색 결과 LOT 리스트 (lotId, 잔여, 가격)
- [[30_모듈별_상세/재고_조회]] → [[30_모듈별_상세/출고_관리]]: BulkSubmitSheet으로 카트 → 출고 신청 직접 전달
- [[30_모듈별_상세/QR_스캔]] → [[30_모듈별_상세/재고_조회]]: 스캔 LOT 번호 → 검색 prefill
