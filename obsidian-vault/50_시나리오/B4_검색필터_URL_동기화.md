# B4 검색 필터 URL 동기화

## 트리거
재고 조회 / 신청 내역 / 결재 수신함에서 검색·필터 입력

## 흐름
1. 사용자가 검색어/필터 입력 → 디바운스 300ms → [[30_모듈별_상세/공통_UI_네비게이션]]
2. URL 쿼리 파라미터 sync (lib/use-sync-query-params.ts) → [[30_모듈별_상세/공통_UI_네비게이션]]
3. 다음 액션 분기:
   - 재고 검색 결과 표시 → [[30_모듈별_상세/재고_조회]]
   - 신청 목록 필터링 → [[30_모듈별_상세/신청_내역]]
   - 결재 수신함 필터링 → [[30_모듈별_상세/결재_수신함]]
4. 뒤로가기 / 새로고침 / URL 공유 시 — 쿼리에서 상태 복원 (검색은 자동 실행 X, 사용자가 수동 트리거)

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/공통_UI_네비게이션]] → 라우터: URL 쿼리 set
- 라우터 → 각 페이지: 쿼리 → 폼 상태 복원
