# A7 자동 로그아웃 5분 전 경고

## 트리거
세션 만료 5분 전 시점 도달 → 상단 배너 노출

## 흐름
1. 로그인 후 세션 타이머 시작 (자동 로그아웃 시점 = 로그인 + N분) → [[30_모듈별_상세/세션_자동로그아웃]]
2. 5분 전 임계 도달 — SessionExpiryBanner 노출 → [[30_모듈별_상세/공통_UI_네비게이션]]
3. 1분 이하 시 빨간 강조 → [[30_모듈별_상세/공통_UI_네비게이션]]
4. 사용자 액션 분기:
   - **로그인 연장** 클릭 → 세션 갱신 (PIN 재입력 없이 토큰 연장) → [[30_모듈별_상세/세션_자동로그아웃]]
   - 무시 → 만료 시 자동 로그아웃 → 로그인 화면 리다이렉트 → [[30_모듈별_상세/PIN_인증]]

## 모듈 간 데이터 흐름
