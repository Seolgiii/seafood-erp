# A3 재고 이동 (보관처 변경)

## 트리거
작업자가 재고 조회에서 LOT을 다른 보관처로 이동 신청 → 관리자 승인

## 흐름
1. 작업자가 재고 조회로 이동 대상 LOT 검색 → [[30_모듈별_상세/재고_조회]]
2. 다건 LOT 카트에 담기 (출고와 동일 카트 UX) → [[30_모듈별_상세/재고_이동]]
3. POST + Idempotency → [[30_모듈별_상세/POST_Idempotency]]
4. 재고이동 레코드 생성 (펜딩) → [[30_모듈별_상세/재고_이동]]
5. 관리자 결재 수신함 검토 → [[30_모듈별_상세/결재_수신함]]
6. 승인 시 — 새 입고관리 레코드 생성 (비고="재고 이동") + 새 LOT 생성 → [[30_모듈별_상세/입고_관리]] / [[30_모듈별_상세/LOT별_재고]]
7. 새 보관처의 시점 비용 적용 → [[30_모듈별_상세/보관처_비용_이력]]
8. 원본 LOT 차감 (재고수량 감소)

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/재고_조회]] → [[30_모듈별_상세/재고_이동]]: 카트 LOT 리스트 + 새 보관처
- [[30_모듈별_상세/재고_이동]] → [[30_모듈별_상세/입고_관리]]: 새 입고관리 레코드 (비고="재고 이동")
- [[30_모듈별_상세/입고_관리]] → [[30_모듈별_상세/LOT별_재고]]: 새 LOT 생성
- [[30_모듈별_상세/재고_이동]] → [[30_모듈별_상세/LOT별_재고]]: 원본 LOT 차감
