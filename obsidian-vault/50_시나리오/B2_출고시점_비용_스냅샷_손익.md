# B2 출고시점 비용 스냅샷 + 손익

## 트리거
출고 승인 시점에 LOT의 비용을 동결(스냅샷)하고 손익 계산

## 흐름
1. 출고 승인 → LOT/입고관리에서 입고시점 비용 조회 → [[30_모듈별_상세/LOT별_재고]] / [[30_모듈별_상세/보관처_비용_이력]]
2. 출고시점 보관처 비용 이력 lookup (출고일자 기준) → [[30_모듈별_상세/보관처_비용_이력]]
3. cost-calc — 입고시점 / 출고시점 / 누적 냉장료 / 입출고비 / 노조비 / 판매원가 / 마진 7필드 산정 → [[30_모듈별_상세/출고_관리]]
4. 출고관리 레코드에 7필드 스냅샷 저장 (반려 시 null 복구 대상) → [[30_모듈별_상세/출고_관리]]
5. 멱등 가드 — 판매원가 > 0 시 차단 (재승인 시 이중 차감 방지) → [[30_모듈별_상세/출고_관리]]

## 모듈 간 데이터 흐름
- [[30_모듈별_상세/LOT별_재고]] → [[30_모듈별_상세/출고_관리]]: 입고시점 비용 + 입고일자
- [[30_모듈별_상세/보관처_비용_이력]] → [[30_모듈별_상세/출고_관리]]: 출고시점 단가
- [[30_모듈별_상세/출고_관리]] → 자기 레코드: 출고시점 비용 7필드 스냅샷
