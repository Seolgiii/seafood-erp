# B1 LOT 생성 시점 비용 적용

## 트리거
입고 승인 시점에 새 LOT가 만들어질 때

## 흐름
1. 입고 승인 → LOT 자동 생성 (일련번호 발급) → [[30_모듈별_상세/입고_관리]] / [[30_모듈별_상세/LOT별_재고]]
2. LOT의 보관처를 키로 보관처 비용 이력 조회 → [[30_모듈별_상세/보관처_비용_이력]]
3. 입고일자 기준 유효 단가 lookup (시점 기반) → [[30_모듈별_상세/보관처_비용_이력]]
4. LOT의 보관처 비용 3필드 set — 냉장료단가 / 입출고비 / 노조비
5. 비용 이력에 매칭 단가 없으면 [INTEGRITY-ALERT] 로그 후 비용 필드 null
