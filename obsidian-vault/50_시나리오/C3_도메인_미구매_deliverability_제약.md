## 흐름 (의사결정 의존 — 임시 fix와 근본 해결 사이의 갈림)
1. Vercel Cron 09:00 KST → Resend API 200 응답 → [[30_모듈별_상세/일일_정산_이메일]]
2. 수신함에 미도착 — 발신 도메인(`onboarding@resend.dev`)과 본문 링크 도메인(`seafood-erp.vercel.app`) mismatch로 spam 트리거
3. **임시 fix 채택** — 본문 CTA 링크 제거 → [[30_모듈별_상세/일일_정산_이메일]]
4. **근본 해결 보류** — 자체 도메인 구매 + 발신 도메인 인증 + 본문 링크 도메인 통일
