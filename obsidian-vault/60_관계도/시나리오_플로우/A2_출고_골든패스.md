## 시퀀스

```mermaid
sequenceDiagram
    participant 작업자
    participant 재고_조회
    participant 출고_관리
    participant 입력_Sanitize_Schema
    participant POST_Idempotency
    participant 결재_수신함
    participant LOT별_재고
    participant 보관처_비용_이력
    participant PDF_생성

    작업자->>재고_조회: 1. 작업자가 재고 조회에서 조건 검색 → LOT 후보 표시
    재고_조회->>출고_관리: 2. 다건 LOT 카트에 담기 (수량·고객 입력)
    출고_관리->>입력_Sanitize_Schema: 3. 입력 sanitize + zod 검증
    입력_Sanitize_Schema->>POST_Idempotency: 4. POST + X-Idempotency-Key
    POST_Idempotency->>출고_관리: 5. 출고 레코드 생성 (펜딩)
    출고_관리->>결재_수신함: 6. 관리자 결재 수신함 검토
    결재_수신함->>출고_관리: 7. 승인 — 출고 멱등 가드 체크 (판매원가>0이면 차단)
    출고_관리->>LOT별_재고: 8. LOT 재고수량 차감 + 입고관리.잔여수량 차감
    LOT별_재고->>보관처_비용_이력: 9. 출고시점 비용 7필드 스냅샷 (cost-calc)
    보관처_비용_이력->>PDF_생성: 10. 출고증 PDF 생성
    Note over PDF_생성: 11. PDF Vercel Blob 업로드
```
