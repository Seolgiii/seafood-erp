# A1 입고 골든패스

> 마지막 갱신: 2026-05-08

## 시퀀스

```mermaid
sequenceDiagram
    participant 작업자
    participant 관리자
    participant 입고_관리
    participant 입력_Sanitize_Schema
    participant POST_Idempotency
    participant 신청_내역
    participant 결재_수신함
    participant LOT별_재고
    participant 보관처_비용_이력
    participant PDF_생성

    작업자->>입고_관리: 1. 작업자가 입고 폼 입력 (품목/규격/미수/보관처/매입처)
    입고_관리->>입력_Sanitize_Schema: 2. 입력 sanitize + zod 검증
    입력_Sanitize_Schema->>POST_Idempotency: 3. POST + X-Idempotency-Key 헤더
    POST_Idempotency->>입고_관리: 4. 입고 레코드 생성 (상태=펜딩, LOT 미생성)
    작업자->>신청_내역: 5. 작업자가 신청 내역에서 확인
    관리자->>결재_수신함: 6. 관리자가 결재 수신함에서 검토
    결재_수신함->>LOT별_재고: 7. 승인 시 — LOT 자동 생성 (일련번호 발급, 영업일 9시 기준)
    LOT별_재고->>보관처_비용_이력: 8. 보관처 비용 이력에서 시점 단가 lookup → LOT 비용 3필드 set
    보관처_비용_이력->>PDF_생성: 9. 입고증 PDF 생성 + LOT QR 임베드
    Note over PDF_생성: 10. PDF Vercel Blob 업로드, 입고관리.PDF필드 set
```
