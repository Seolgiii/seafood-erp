# 입고 골든패스 시나리오

> 시나리오 [[A1_입고_골든패스]]를 두 가지 다이어그램으로 시각화.
> 마지막 갱신: 2026-05-08

## 단계별 플로우 (시퀀스)

```mermaid
sequenceDiagram
    participant 워커 as 워커
    participant 입고 as 입고_관리
    participant 결재 as 결재_수신함
    participant LOT as LOT별_재고
    participant 비용 as 보관처_비용_이력
    participant PDF as PDF_생성

    워커->>입고: 입고 신청
    입고->>결재: 결재 요청 (승인 대기)
    결재-->>워커: 알림
    워커->>결재: 승인 (워커/마스터)
    결재->>LOT: LOT 생성 트리거
    LOT->>비용: 보관처 비용 lookup
    비용-->>LOT: 3필드 (냉장료/입출고비/노조비)
    LOT->>PDF: 입고증 PDF 발행
    PDF-->>워커: 입고증 (QR 포함, 통일안)

    Note over LOT: QR이 LOT의 평생 식별자
    Note over PDF: QR 스캔 시 /lot/{번호}로
```

## 의존 관계도 (같은 시나리오를 다른 각도)

```mermaid
graph LR
    트리거[워커 입고 신청] --> 입고
    입고 --> 결재
    결재 -->|승인| LOT
    LOT --> 비용
    비용 -->|3필드| LOT
    LOT --> PDF
    PDF -->|QR 포함| 출력[입고증 발행]

    classDef start fill:#90ee90
    classDef ending fill:#ffd700
    class 트리거 start
    class 출력 ending
```
